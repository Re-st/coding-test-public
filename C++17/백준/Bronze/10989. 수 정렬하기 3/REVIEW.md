## 1) 요약
제출된 코드는 **계수 정렬(Counting Sort)** 알고리즘을 사용해 문제를 해결합니다. 입력으로 주어지는 수의 범위(1 ~ 10,000)가 크지 않다는 점을 활용하여, 각 숫자의 등장 횟수를 저장하는 `count` 배열을 만듭니다. 이후 배열을 순회하며 각 숫자를 등장 횟수만큼 반복 출력하여 정렬된 결과를 얻습니다.

- **시간 복잡도:** O(N + K) (N: 수의 개수, K: 수의 최대값, 여기서는 10,000)
- **공간 복잡도:** O(K)

## 2) 시간 / 공간 복잡도 점검
문제의 제약 조건은 `N`이 최대 10,000,000이고, 입력되는 수는 10,000 이하의 자연수입니다.

- **시간 복잡도**: O(N + K)는 약 10,000,000 + 10,000 = 10,010,000번의 연산에 해당합니다. 이는 1초 안에 충분히 실행 가능한 매우 효율적인 접근입니다. 만약 `std::sort`와 같은 비교 기반 정렬(O(N log N))을 사용했다면, 10^7 * log(10^7) ≈ 10^7 * 24 정도의 연산으로 시간 초과가 발생했을 수 있습니다.
- **공간 복잡도**: O(K)는 `int count[10001]` 배열의 크기에 해당하며, 약 `10001 * 4 bytes` ≈ 40KB 입니다. 이는 메모리 제한(문제 미명시, 보통 128~512MB)에 비해 매우 적은 양으로, 안전합니다.

결론적으로, 문제의 제약 사항에 가장 최적화된 알고리즘을 선택하셨습니다.

## 3) 구현/버그 리스크
현재 코드는 간결하고 명확하여 버그 발생 위험이 매우 낮습니다. 몇 가지 포인트를 짚어보자면 다음과 같습니다.

- **배열 크기**: `count[10001]`은 10,000이라는 값을 인덱스로 사용하기에 안전합니다. C++ 배열은 0부터 시작하므로 크기가 `10001`이면 인덱스는 `0`부터 `10000`까지 사용할 수 있습니다.
- **배열 초기화**: `int count[10001] = { 0 };` 구문을 통해 배열의 모든 원소가 0으로 확실하게 초기화됩니다. 이 부분이 누락되면 쓰레기 값으로 인해 오작동할 수 있는데, 잘 처리되었습니다.
- **입출력 속도**: `N`이 매우 크므로 `ios::sync_with_stdio(false); cin.tie(NULL);`를 사용하여 입출력 속도를 최적화한 것은 필수적이며, 올바르게 적용되었습니다.

전반적으로 위험 요소가 거의 없는 깔끔한 구현입니다.

## 4) 엣지 케이스 체크리스트
- **최소 입력**: N=1일 때, 정상적으로 동작합니다.
- **최대 입력**: N=10,000,000일 때, 시간/메모리 복잡도상 안전합니다.
- **중복 값**: 모든 입력값이 동일할 경우(예: 10,000,000개의 7), `count[7]`에 10,000,000이 저장되고 7이 올바른 횟수만큼 출력됩니다.
- **경계 값**: 입력으로 1과 10,000이 주어질 때, 각각 `count[1]`과 `count[10000]`에 올바르게 반영되므로 문제가 없습니다.
- **입력값이 한쪽으로 쏠린 경우**: 모든 수가 1~100 사이에만 분포하더라도 `for`문은 10,000까지 돌지만, `count` 값이 0인 부분은 `while` 루프에 진입하지 않으므로 성능에 큰 영향은 없습니다.

현재 풀이는 위 모든 엣지 케이스를 잘 처리할 수 있습니다.

## 5) 리팩토링/가독성 개선 (선택)
코드가 매우 간결하여 크게 개선할 부분은 없지만, 약간의 수정을 통해 의도를 더 명확히 할 수 있습니다.

1.  **상수 사용**: 코드에 사용된 `10001`, `10000`과 같은 매직 넘버(magic number)를 상수로 정의하면 가독성과 유지보수성이 향상됩니다.

    ```cpp
    #include <iostream>

    const int MAX_VAL = 10000;

    int main() {
        // ... (입출력 최적화)
        int n, now;
        std::cin >> n;
        int count[MAX_VAL + 1] = { 0 };
        while (n--) {
            std::cin >> now;
            count[now]++;
        }
        for (int i = 1; i <= MAX_VAL; i++) {
            while (count[i]--) {
                std::cout << i << '\n';
            }
        }
        return 0;
    }
    ```

2.  **불필요한 헤더 제거**: 코드에서 `<vector>` 헤더를 사용하지 않으므로 제거하는 것이 좋습니다.
