## 즉시 수정 액션(최대 4개)
- 1. **(가독성) `dfs` 함수 내 백트래킹 로직을 명확히 하세요.** 현재 코드는 `return` 직전마다 `visited[v] = false`를 호출하는데, 함수 끝에서 한 번만 처리하는 것이 더 깔끔하고 일반적인 백트래킹 구조입니다.
  ```cpp
  bool dfs(int v, int depth) {
      if (depth == 5) return true;
      visited[v] = true;
      for (int i : e[v]) {
          if (!visited[i] && dfs(i, depth + 1)) {
              return true; // 성공 시 즉시 전파
          }
      }
      visited[v] = false; // v에서 시작한 모든 경로 탐색 후 상태 원복
      return false;
  }
  ```
