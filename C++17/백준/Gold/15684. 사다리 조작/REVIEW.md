## 한 줄 결론
- (이 문제는) **백트래킹** / 복잡도: O((NH)³ * NH) / 최소 개수(1, 2)를 찾고도 즉시 종료하지 않아 더 큰 답이 나올 수 있는 실수

## 더 빠른/더 안전한 풀이(필수)
- 현 접근(완전탐색)은 맞지만, 재귀(DFS)를 이용한 백트래킹이 훨씬 안전합니다.
  - 아이디어: `dfs(추가한 사다리 수, 탐색 시작 위치)` 함수로 k개의 사다리를 놓는 모든 조합을 탐색합니다.
  - 복잡도: O((NH)³ × NH)로 동일.
  - 왜 이게 더 좋은지: 3중 이상의 for문보다 조합 생성이 직관적이고, 정답 발견 시 즉시 종료하는 로직 구현이 쉬워 버그가 줄어듭니다.

## 내가 지금 틀리기 쉬운 포인트 TOP3
- 1) **종료 조건**: 2개로 해결 가능한데 3을 답으로 낼 수 있습니다. 2개짜리 답(`ans=2`)을 찾고도 루프를 계속 돌다가, 다른 분기에서 3개짜리 답을 찾아 `ans`를 덮어쓰기 때문입니다. **최소값**을 찾으면 즉시 프로그램을 종료해야 합니다.
- 2) **경계/인덱스**: `all_identical` 함수의 `for (int j = 1; j < n; j++)`은 마지막 `n`번 세로선을 확인하지 않아 `res[n] != n`인 경우를 놓칩니다. `j <= n`으로 수정해야 합니다.
- 3) **성능 함정**: `for (a2 = a1...) for(b2 = 1...)` 형태의 루프는 사다리 놓는 순서만 다른 동일 조합을 중복 검사하여 비효율적입니다. 탐색 시작 위치를 이전 위치 다음부터로 명확히 지정해야 합니다. (예: `b2 = (a1 == a2 ? b1 + 1 : 1)`)

## 즉시 수정 액션(최대 4개)
- `all_identical` 함수 내 `for (int j = 1; j < n; j++)`를 `for (int j = 1; j <= n; j++)`로 수정하세요.
- 1, 2, 3개 사다리를 추가해 성공하는 각 지점에서, `ans` 변수 할당 대신 `cout`으로 답을 출력하고 `return 0;`로 즉시 종료하세요.
- `collide` 함수에서 `lad[a][b-1]` 접근 시 `b=1`인 경우 0번 인덱스에 접근합니다. `lad` 배열이 넉넉해 문제는 없지만, 명시적인 경계 처리가 안전합니다.
- (권장) 3중 for문 구조를 재귀 함수로 리팩토링하여 코드 구조를 단순화하고 버그 가능성을 줄이세요.

```REVIEW_EXAMPLE
# 제목: 재귀(DFS)를 이용한 백트래킹 풀이
- 적용 조건: 현재처럼 추가할 개수(k)가 3 이하로 정해진 조합 탐색 문제에 가장 이상적입니다.
- 핵심 아이디어: 추가할 사다리 개수(0~3)를 목표로 설정하고, DFS로 해당 개수만큼 사다리를 놓는 조합을 탐색합니다. 목표 개수만큼 사다리를 놓았다면 성공 여부를 체크하고, 성공 시 즉시 프로그램을 종료합니다.
- 복잡도: O((NH)³ × NH)
- 구현 스케치(C++):
int ans = -1;
void dfs(int count, int target, int start_r) {
    if (count == target) {
        if (all_identical()) {
            ans = target;
        }
        return;
    }

    for (int i = start_r; i <= h; ++i) {
        for (int j = 1; j < n; ++j) {
            if (lad[i][j] || lad[i][j - 1] || lad[i][j + 1]) continue;
            lad[i][j] = true;
            dfs(count + 1, target, i); // 다음 탐색은 현재 행부터 시작해 중복 방지
            lad[i][j] = false;
            if (ans != -1) return; // 하위 탐색에서 답을 찾았으면 즉시 복귀
        }
        if (ans != -1) return;
    }
}

int main() {
    // ... 입력 처리 ...
    for (int i = 0; i <= 3; ++i) {
        dfs(0, i, 1);
        if (ans != -1) {
            cout << ans;
            return 0;
        }
    }
    cout << -1;
}
```
