## 한 줄 결론
- (이 문제는) **시뮬레이션** / 복잡도: O(단계 수 * N) / 제일 위험한 실수는 **로봇 충돌 검사 로직 오류**입니다.

## 더 빠른/더 안전한 풀이(필수)
- 아이디어: 로봇의 위치를 `deque`가 아닌 `bool on_robot[2N]` 배열로 따로 관리합니다.
- 복잡도: O(단계 수 * N) (기존과 동일)
- 왜 이게 더 좋은지: 로봇의 존재 여부와 충돌을 O(1)에 명확히 확인할 수 있어, 복잡한 `deque` 조작 없이 버그 발생 가능성을 크게 줄이고 코드가 직관적으로 바뀝니다.

## 내가 지금 틀리기 쉬운 포인트 TOP3
- 1) **로봇 이동 순서 및 충돌**: 가장 먼저 올라간 로봇(내리는 위치 `N-1`에 가까운)부터 이동해야 합니다. 현재 `deque`를 조작하며 다른 로봇 위치를 확인하는 `robot.front() != r + 1` 로직은 일부 로봇만 확인하므로 충돌을 제대로 감지할 수 없습니다.
- 2) **상태 업데이트 순서 혼재**: 벨트 회전과 로봇의 자체 이동이 하나의 루프 안에서 `++r`로 섞여있어 로직이 복잡하고 실수하기 쉽습니다. (1)벨트+로봇 전체 회전, (2)로봇 자체 이동, (3)새 로봇 투입 단계를 명확히 분리해야 합니다.
- 3) **경계 조건**: 로봇이 내리는 위치(`N-1`)에 도달하면 즉시 하차해야 합니다. 벨트 회전 직후와 로봇의 자체 이동 직후, 두 번의 하차 시점을 모두 명시적으로 처리해야 누락이 없습니다.

## 즉시 수정 액션(최대 4개)
- 1. 로봇 위치 관리를 `deque<short> robot` 대신 `bool on_robot[2*N]` 배열로 변경하세요.
- 2. 시뮬레이션의 각 단계를 분리된 `for`문으로 구현하세요 (벨트 회전 → 로봇 이동 → 로봇 투입).
- 3. 로봇 이동 로직은 가장 먼저 올라간 로봇부터 처리하도록 `for (int i = N-2; i >= 0; --i)` 형태로 구현하세요.
- 4. 로봇이 내리는 위치(`N-1`)에 있는지 확인하는 로직을 벨트 회전 직후와 로봇 이동 직후, 두 번 실행하여 즉시 하차 규칙을 준수하세요.

```REVIEW_EXAMPLE
# 제목: 더 안전하고 직관적인 풀이 (로봇 위치 배열 활용)
- 적용 조건: 현재 풀이는 로봇 위치 추적이 복잡하여 버그 위험이 높으므로, 이 대안이 항상 더 유리합니다.
- 핵심 아이디어: `deque`로는 벨트의 내구성만 관리하고, 로봇의 위치는 `bool robot_on[2*N]` 배열로 별도 관리합니다. 이를 통해 로봇의 존재 여부와 이동 가능 여부를 `O(1)`에 명확하게 확인할 수 있습니다.
- 복잡도: O(단계 수 * N)
- 구현 스케치(의사코드):
vector<bool> robot_on(2 * N, false);
while (zero_count < K) {
    step++;

    // 1. 벨트와 로봇 함께 회전
    belt.push_front(belt.back());
    belt.pop_back();
    for (int i = N - 2; i >= 0; --i) {
        robot_on[i + 1] = robot_on[i];
    }
    robot_on[0] = false;
    if (robot_on[N - 1]) robot_on[N - 1] = false; // 회전 후 즉시 하차

    // 2. 로봇 스스로 이동 (가장 먼저 올라간 로봇부터)
    for (int i = N - 2; i >= 0; --i) {
        if (robot_on[i] && !robot_on[i + 1] && belt[i + 1] >= 1) {
            robot_on[i] = false;
            robot_on[i + 1] = true;
            if (--belt[i + 1] == 0) zero_count++;
        }
    }
    if (robot_on[N - 1]) robot_on[N - 1] = false; // 이동 후 즉시 하차

    // 3. 새 로봇 올리기
    if (belt[0] > 0) {
        robot_on[0] = true;
        if (--belt[0] == 0) zero_count++;
    }
}
```
